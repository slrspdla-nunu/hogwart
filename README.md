# 🪄 호그와트 레거시 (Hogwarts Legacy) 팬사이트

게임 **호그와트 레거시**를 주제로 제작한 반응형 팬사이트입니다.
바닐라 **HTML · CSS · JavaScript** 만으로 구현했으며, 마법 세계 분위기를 살린 인터랙션과
메인 1개 + 서브 4개 페이지로 구성된 포트폴리오 프로젝트입니다.

> ⚠️ 비상업적 팬 프로젝트입니다. 커뮤니티(방명록) 등 창작 콘텐츠에는 원작 등장인물을 사용하지 않았습니다.
> HOGWARTS LEGACY, WIZARDING WORLD 및 관련 상표·저작권은 Warner Bros. Entertainment Inc. 및 J.K. Rowling에게 있습니다.

---

## 📄 페이지 구성

| 페이지 | 파일 | 설명 |
|---|---|---|
| **메인** | `index.html` | 히어로, 입학 편지(About), 소개 카드, **기숙사 배정 테스트**, 마법 인터랙션 |
| **주문 도감실** | `gallery.html` | 스크린샷 / 영상 / 컨셉아트 탭 갤러리 |
| **호그와트 학업 가이드** | `guide.html` | 책 모양 UI + FAQ 아코디언 (책갈피 탭 전환) |
| **예언자 일보** | `prophet.html` | 소식 목록 + 기사 상세 보기(이전/다음 글) |
| **스리 브룸스틱스 방명록** | `guestbook.html` | 게시글 커뮤니티 + 인기 게시물 + 글 작성 모달 |

---

## ✨ 주요 기능

- **기숙사 배정 테스트** — 6문항 응답으로 기숙사 결정, 결과·상세 뷰, 전체 기숙사 보기
- **마법 인터랙션 (메인)** — 골든 스니치 잡기, 루모스/녹스 조명 토글, 펠릭스 펠리시스(행운의 물약)
- **책 UI + FAQ 아코디언** — 학업 가이드 페이지의 책갈피 전환·펼침
- **커뮤니티 방명록** — 게시글 카드, 인기 게시물, 글 작성 모달(즉시 반영)
- **기사 뷰어** — 예언자 일보 목록/상세 전환, 이전·다음 글 이동
- **스크롤 등장 애니메이션**, 히어로 패럴랙스, 섹션 도트 내비게이션

## 📱 반응형 디자인

- 데스크톱 · 태블릿 · 모바일 대응 (전 구간 가로 스크롤 없음)
- **모바일 햄버거 메뉴** (좁은 화면에서 상단 메뉴 토글)
- 화면별 전용 레이아웃 (예: 학업 가이드 책은 모바일/태블릿에서 세로형 단일 책으로 전환)
- **디자인 토큰(타입 스케일·굵기·여백)** 기반 텍스트 규칙 통일, 한글 단어 단위 줄바꿈(keep-all)

## 🛠️ 기술 스택

- **HTML5**, **CSS3** (Flexbox / Grid / 커스텀 프로퍼티 · 미디어쿼리)
- **Vanilla JavaScript** (프레임워크·라이브러리 없음)
- 웹폰트: Cinzel, Noto Serif KR, Noto Sans KR

## 🚀 실행 방법

정적 사이트라 `index.html`을 브라우저로 열면 됩니다.
로컬 서버로 확인하려면:

```bash
# Python
python -m http.server 8000
# 이후 http://localhost:8000 접속
```
